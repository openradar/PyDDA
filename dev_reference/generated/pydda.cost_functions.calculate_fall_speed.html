pydda.cost_functions.calculate_fall_speed¶

pydda.cost_functions.calculate_fall_speed(grid, refl_field=None, frz=4500.0)[source]¶

Estimates fall speed based on reflectivity.

Uses methodology of Mike Biggerstaff and Dan Betten

Parameters
Grid: Py-ART Grid

Py-ART Grid containing reflectivity to calculate fall speed from

refl_field: str

String containing name of reflectivity field. None will automatically determine the name.

frz: float

Height of freezing level in m

Returns
3D float array:

Float array of terminal velocities
