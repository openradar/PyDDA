pydda.constraints.download_needed_era_data#

pydda.constraints.download_needed_era_data(Grid, start_date, end_date, file_name)[source]#

This function will download the ERA interim data in the region specified by the input Py-ART Grid within the interval specified by start_date and end_date. This is useful for the batch processing of files since the ECMWF API is limited to 20 queued requests at a time. This is also useful if you want to store ERA interim data for future use without having to download it again.

You need to have the ECMWF API and an ECMWF account set up in order to use this feature. Go to this website for instructions on installing the API and setting up your account:

https://confluence.ecmwf.int/display/WEBAPI/Access+ECMWF+Public+Datasets
