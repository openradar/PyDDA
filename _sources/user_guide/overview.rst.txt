Overview
==========================

Within this userguide, we will detail the entire multi-doppler workflow, including:

* How to read radar data with Py-ART
* How to clean your radar data
* How to grid your radar data to cartesian coordinates
* How to run PyDDA and tune the parameters
* How to visualize your PyDDA output

If you are interested in adding a section to the user guide,
please use the template.rst file in the documentation.
