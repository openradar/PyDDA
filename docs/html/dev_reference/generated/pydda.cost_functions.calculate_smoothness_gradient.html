pydda.cost_functions.calculate_smoothness_gradient¶

pydda.cost_functions.calculate_smoothness_gradient(u, v, w, Cx=1e-05, Cy=1e-05, Cz=1e-05, upper_bc=True)[source]¶

Calculates the gradient of the smoothness cost function by taking the Laplacian of the Laplacian of the wind field.

All grids must have the same grid specification.

Parameters:

u: Float array

Float array with u component of wind field

v: Float array

Float array with v component of wind field

w: Float array

Float array with w component of wind field

Cx: float

Constant controlling smoothness in x-direction

Cy: float

Constant controlling smoothness in y-direction

Cz: float

Constant controlling smoothness in z-direction
Returns:

y: float array

value of gradient of smoothness cost function
